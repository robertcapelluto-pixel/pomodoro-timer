# Pomodoro Timer

A simple, single-file Pomodoro timer that runs in the browser. No installs, no dependencies — just open the file and start working.

## Features

- 25-minute work sessions that automatically switch to 5-minute breaks
- Start/Stop button to pause and resume the timer at any time
- Reset button to return to the beginning of a work session
- Session counter that tracks how many Pomodoros you've completed
- Color accent shifts between work mode (red) and break mode (teal)
- Live countdown reflected in the browser tab title
- Background image slideshow that cycles every 5 seconds

## How to use

1. Clone or download this repository
2. Open `pomodoro.html` in any modern web browser
3. Press **Start** to begin the timer
4. Work until the timer hits zero — it will automatically switch to a break
5. After the break, it switches back to a work session
6. Press **Stop** to pause, or **Reset** to start over

No server required. Everything runs locally in the browser.
